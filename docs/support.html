Frequently Asked Questions

What is Snapcast?

Snapcast is a multi-room client-server audio player. You need a Snapcast server running on your network to use SnapCTRL.

Why can't I find my server?

  • Ensure your Snapcast server is running and on the same network
  • Check that your server has mDNS/Bonjour enabled
  • Try adding the server manually using its IP address
  • Verify that SnapCTRL has local network permission in iOS Settings

Why is there no audio?

  • Check that the server is streaming audio (source is playing)
  • Verify volume is not muted in the app or on the server
  • Ensure you're assigned to a group with an active stream
  • Check that iOS volume is turned up

Does it work with AirPlay?

SnapCTRL receives audio from Snapcast servers, not from AirPlay sources. If your Snapcast server has an AirPlay source configured, it will work. Do not use AirPlay output from your iOS device to a Snapcast client as this creates a feedback loop.
